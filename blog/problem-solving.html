What was the problem I was facing?

The problem I was blocked in was the concatMyArray. I tried to solve this problem by entering the strings through a for loop and found that I was stuck at pushing a string to an empty variable string.

"2: failed - concatMyArray should return a string where all the strings in the given array have been joined together."  I kept trying various things but it kept showing in the test

Recieved: "" Expected: was "zoomzoom!"

After receiving the error message above, I googled how to join an array of string into one string. Immediately came up with the join() method. Read through the documentation and realised it was actually a lot easier than expected. You just need to add the array then join()  and '' without spaces like example below.  array.join('') The outcome will be "thisisastring" and if you typed in array.join(' ') this will create spaces in between each word.

I was frustrated because if googled first how to join array of strings into one i could've completed that challenge a lot sooner.

I learned before starting the challenge its a good idea to check if there's an actually method to make the code easier to write without too many lines and also read the instructions properly.

Elegantly solved a problem

I was able to solve the SuperFizzBuzz fairly quickly. This is only because I've done this many times in different formats and requirements. It was fun being able to solve this one with ease.

I usually break it down using pseudocode then wrote out each if and else statements so that its much easier to manage. Once I wrote the code for the first fizzbuzz function and passed then I copied and paste the existing code into a for loop. I couldn't figure out why it wasn't looping through and return the correct outcome. So I decided to use my fizzbuzz function inside the loop passing each of the array item, created an empty array and pushed into new array and returned the new array with fizz buzz included.

I felt great being able to complete this challenge quickly without too many problems. Although after completing it realised why my previous test with super_fizzbuzz didn't work and that was because I never created an empty array and returned it.

I learned how to call another function within a function to cut down on length of code and make more readable.
